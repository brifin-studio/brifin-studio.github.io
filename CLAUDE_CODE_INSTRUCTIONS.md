# Инструкция для Claude Code

Замени текущий `index.html` на новую версию из этого чата.
Файл уже полностью готов и проверен на синтаксические ошибки.

## Команда

```bash
# Находясь в папке C:\Users\User\Дизайнер\
# Просто замени index.html на скачанный из чата файл, затем:

git add index.html
git commit -m "redesign: новый экран входа, сайдбар дизайнера, модалка авторизации"
git push
```

## Что изменилось

### Экран входа
- Убраны две карточки "Дизайнер/Заказчик"
- Одна большая карточка "Поставить задачу" по центру
- Кнопка "Дизайнер" — маленькая, левый верхний угол
- Клиент вводит имя один раз — потом запоминается через localStorage

### Авторизация дизайнера
- Модалка 2 шага с индикатором прогресса
- Шаг 1: выбор из списка, inline-ошибка если не выбрал
- Шаг 2 (только первый вход): фото + отображаемое имя + цвет карточек
- Повторный вход: localStorage помнит — сразу в доску

### Интерфейс дизайнера
- Фиксированный сайдбар 200px вместо шапки
- Профиль сверху кликабелен → редактирование
- Навигация: Все задачи / Мои задачи
- Выйти внизу
- Анимация при переключении фильтра

### Редактирование профиля
- Смена имени, фото, цвета
- Сохраняется в Firebase + localStorage
